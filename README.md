# VØRTEX - Tienda de Streetwear

Tienda de streetwear en línea con estilo urbano y calidad premium.

## Características

- 🛍️ Catálogo de productos
- 🛒 Carrito de compras
- 💳 Sistema de pago y checkout
- 👤 Sistema de login
- 📱 Diseño responsive
- 🎨 Interfaz moderna

## Estructura del Proyecto

```
├── index.html          # Página principal
├── catalogo.html       # Catálogo de productos
├── producto.html       # Detalle de producto
├── carrito.html        # Carrito de compras
├── checkout.html       # Checkout
├── pago.html          # Página de pago
├── confirmacion.html   # Confirmación de compra
├── login.html         # Login
├── contacto.html      # Contacto
├── novedades.html     # Novedades
├── styles.css         # Estilos globales
└── assets/
    └── images/        # Imágenes del proyecto
```

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/chuchobck/IHC.git
cd IHC
```

2. Abre `index.html` en tu navegador

## Despliegue en Vercel

1. Conecta tu repositorio GitHub a Vercel
2. Vercel detectará automáticamente tu sitio HTML estático
3. Tu sitio estará disponible en: `https://ihc.vercel.app`
